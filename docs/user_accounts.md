## User Accounts

There are three types of user accounts on Satori-CI:

----------------------------------------------------------------------------
| Name |                       Daily Executions | Restrictions             |
----------------------------------------------------------------------------
| Free |                                     10 | - No internet connection |
|      |                                        | - No monitor with cron   |
|      |                                        | - Lower priority         |
| User |                                   1000 |                          |
| Pro  |                                   5000 |                          |
